Employee Performance Prediction
-----------------------------------------
A Machine Learning web application that predicts whether an employee is a High, Average, or Low Performer based on key HR metrics.
-> Project Overview
This project uses a Random Forest Classifier to predict employee performance using historical HR data.
The model is trained on features like:
 1. Attendance Rate
 2.Years at Company
 3.Training Hours
 4.Previous Rating
 5.Overtime Hours
The application includes:
 1. Machine Learning model
 2. Flask web interface
 3. Saved model using Pickle
